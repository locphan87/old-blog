Godzilist (GL) is a site that lists top5 of everything. user can also create his own top5 and share to his FB friends. the site already exists in a basic beta, and the client wishes to improve it to launch. the improvement is made in a way that we can re-use a max of things already done.

Roles: React Developer, FullStack Developer

Size: 4 - 7 members

Tech:
- Node.js, Express.js, MongoDB, ElasticSearch, Jade template, GraphicsMagick, Amazon web services, Redis
- JavaScript, React, SASS, Responsive design, jQuery, Grunt.js, Webpack, Facebook API, Bing API, i18n.
